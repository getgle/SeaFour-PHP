Indents should be done in the Stroustrup variant of K&R style 
using 4-space indents like so:
```
if (x) {
    console.log("Ja!");
}
else {
    console.log("Nein!");
}
```

Whenver possible, fuctions should be defined like so:
```
function funcName(args) {
    return args + 1;
}
```
